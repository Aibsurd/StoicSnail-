## Операционные правила

### Память
- Хочешь запомнить → пиши в файл
- Ментальные заметки не выживают
- **Решения → MEMORY.md**
- **События → memory/YYYY-MM-DD.md**
- **Проекты → PROJECTS.md**

### Projects
- Начинаешь проект → добавить в PROJECTS.md
- Закрываешь задачу → обновить PROJECTS.md
- Значимое изменение → записать в INTENTS.md

### Исследования
- Использовать `scripts/deep_research.sh`
- Результаты → в `research/`
- Старые → в `archive/research/`

## Управление контекстом (SESSION.md)

**Проблема:** Контекстное окно ограничено. При достижении 100% — ошибка или обрезание.

**Решение OpenClaw:**
- Auto-compaction при приближении к лимиту
- Memory flush перед compaction (я записываю важное в файлы)
- Session pruning для tool results

**Настроено:**
- `compaction.mode: safeguard` — защищает recent context
- `reserveTokens: 30000` — место для генерации
- `keepRecentTokens: 80000` — минимум recent context
- `memoryFlush: enabled` — записываю перед сжатием

**Команды:**
| Команда | Когда |
|---------|-------|
| `/compact` | Принудительная compaction |
| `/status` | Показать % context |
| `/new` | Чистый лист, новая сессия |
| `/reset` | То же |

**Стратегия:**
- Long session → `/compact` периодически
- Новый проект → `/new`
- Важное → в файлы (не в память контекста)
