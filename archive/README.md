# Archive

> Старые данные. Можно посмотреть но не нужно держать в активном контексте.

---

## Структура

```
archive/
└── research/
    └── YYYYMMDD_HHMMSS_topic/
        └── report.md
```

---

## Содержимое

| Директория                                  | Тема                      | Дата       |
| ------------------------------------------- | ------------------------- | ---------- |
| `20260321_150208_AI_agents_autonomous_2026` | AI agents autonomous 2026 | 2026-03-21 |

---

## Правила

- **Не читать при старте** — это архив
- **Искать только если нужно** — по конкретному запросу
- **Удалять мусор** — partial results, failed attempts

---

## Автоочистка

Исследования старше 30 дней — удалять если не релевантны.

```
find archive/ -type d -mtime +30 -exec echo "Would remove: {}" \;
```
