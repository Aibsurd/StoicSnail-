# TIME.md — Время и часовой пояс

## Источники времени

### Рантайм (приоритет)
Система инжектирует текущее время в system prompt при каждом запросе:
```
Current time: Saturday, March 21st, 2026 — 7:42 PM (UTC) / 2026-03-21 19:42 UTC
```

### Проверка вручную
```bash
date '+%Y-%m-%d %H:%M %Z'
# → 2026-03-21 19:44 UTC

date '+%A, %B %d, %Y — %I:%M %p (%Z)'
# → Saturday, March 21, 2026 — 07:44 PM (UTC)
```

### session_status
Команда `/status` или `session_status` показывает время сессии.

## Часовой пояс

| Параметр | Значение |
|----------|----------|
| **Zone** | UTC |
| **File** | /etc/timezone → Etc/UTC |
| **Настройка** | Все события логируются в UTC |

## Зачем это нужно

1. **Контекст поиска** — свежие новости, события
2. **Логирование** — датировать записи в memory/
3. **Cron jobs** — понимать расписание
4. **Ориентация** — "сегодня", "вчера", "неделю назад"

## Day boundary

Сессия началась в субботу 2026-03-21. Следующий день — воскресенье 2026-03-22.
